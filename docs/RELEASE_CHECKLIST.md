## ⚠️ 常见问题排查

### **Q1: 云函数调用失败 "cloud function not found"**

```
原因：云函数未上传或命名不一致
解决:
  1. 确认 cloudfunctions 目录下有对应的 index.js 和 package.json
  2. 在微信开发者工具中上传云函数
  3. 检查 project.config.json 中的 cloudfunctionRoot 配置
```

### **Q2: MySQL 连接超时 "Malformed communication packet"**

```
原因：网络连接不稳定或超时时间不足
解决:
  1. 检查腾讯云安全组是否放行 27780 端口
  2. 增加连接超时时间（已在代码中设置为 15 秒）
  3. 使用连接池复用连接（已在代码中实现）
```

### **Q3: 数据统计页面空白**

```
原因：API 返回数据格式错误或图表库未加载
解决:
  1. F12 打开调试面板查看 console.log
  2. 检查 charts.min.js 是否正确引入
  3. 验证返回数据的字段名是否与代码一致
```

### **Q4: 授权弹窗无法关闭**

```
原因：auth-dialog 组件未正确处理取消事件
解决:
  1. 检查 components/auth-dialog/index.wxml 按钮绑定
  2. 确保 onClose 方法正确实现
  3. 验证组件的 hidden 属性是否正确更新
```
