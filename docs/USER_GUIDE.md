## 1. 小程序功能总览

| 标签页 | 功能 | 说明 |
|--------|------|------|
| 📖 **学习** | 首页 | 复习队列、今日推荐单词、学习统计概览 |

| ⚙️ **设置** | 用户设置 | 每日学习目标、听写设置、复习参数 |

## 2. 学习流程

### 2.1 日常学习

```
1. 打开小程序 → 首页
2. 查看"待复习单词"队列
3. 点击"开始复习" → 进入闪卡模式
4. 查看单词 → 思考 → 点击"显示答案"
5. 自评质量 → 自动进入 SM-2 间隔重复
6. 复习完成 → 回家页查看进度
```

### 2.2 听写练习

```
1. 底部导航 → 进入听写页面
2. 系统播放单词音频（或显示提示）
3. 输入拼写
4. 系统判断对错
5. 错误单词自动加入复习队列
```

### 2.3 使用单词集

```
1. 底部导航 → 单词集
2. 点击"+"创建新单词集
3. 进入单词集详情 → 搜索添加单词
4. 点击"开始学习" → 以闪卡模式学习该集合
5. 学习进度自动记录
```

## 3. 数据说明

| 数据 | 存储位置 | 说明 |
|------|----------|------|
| 单词库 | 云端 MySQL | 初中/高中词汇，按年级和教材分类 |
| 学习进度 | 云端 MySQL | 用户独享，通过微信 OpenID 隔离 |
| 用户设置 | 云端 MySQL | 每日学习数量、学习模式偏好 |
| 用户身份 | 微信自动注入 | 无需登录，云托管自动识别 |

## 4. 常见问题

### Q: 需要登录吗？
A: 不需要。微信用户身份由云托管自动识别，打开即用。

### Q: 换设备后数据还在吗？
A: 在的。数据存储在云端，使用同一微信账号即可恢复。

### Q: 单词库有多少单词？
A: 数千个初中/高中英语单词，按年级和教材版本分类。

### Q: 如何调整每日学习量？
A: 进入「设置」页面，可以调整每日新词数和复习数。
