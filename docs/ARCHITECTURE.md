## 2. 通信方式

| 方式 | 说明 | 优势 |
|------|------|------|
| `wx.cloud.callContainer()` | 微信内网私有协议 | 无需域名、自动注入 OpenID、速度快 |
| `X-WX-SERVICE: 'express-yoq0'` | 指定云托管服务名 | 必须传，否则报 -601031 |
| 环境 ID: `prod-d2g668gku1c36b430` | 云托管环境 | 在 callContainer.config.env 中指定 |

## 4. 认证流程

```
用户打开小程序
    ↓
wx.cloud.init() 初始化云托管
    ↓
页面发起 API 请求 (requestLib.request)
    ↓
wx.cloud.callContainer() 走微信内网
    ↓
云托管网关自动注入 x-wx-openid 头
    ↓
Express authMiddleware 读取 req.headers['x-wx-openid']
    ↓
注入 req.openid，后续查询以 openid 隔离数据
```
